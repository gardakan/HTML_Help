Changing a Price Sheet


 

From Price Sheet draft detail screen:

From the bounce bar, select the Change option.

Cursor focus will be on the price sheet type radio group.

  • A Draft can only be changed to a Proposal; an error message will be displayed if an attempt to change it to a Price Sheet is made. Move the cursor from the Draft position to the Proposal position and select it by pressing the spacebar. Press Enter and select the "Yes" option to complete the change from Draft to Proposal. Making this selection will exit the edit screen and bring focus back to the bounce bar.
      NOTE: Selecting the "No" value will return focus to the edit screen and allow the user to change additional fields.








  • A Proposal can either be downgraded to a Draft, or upgraded to a Price Sheet.
    • To downgrade from a proposal to a draft, move the cursor from the Proposal position to the Draft position and select it with the spacebar. Press enter and select "Yes" on the pop-up screen to complete the process.
      NOTE: Selecting "No" will return focus to the edit screen.








    • Please refer to Upgrading A Proposal To A Price Sheet section for specific instructions on how to upgrade from a Proposal to a Price Sheet.

The user has the option of editing the remaining price sheet fields. Please refer to the section on adding a Price Sheet for reference. Please note that the Ship Range fields can not be edited on an existing price sheet - if a different ship range is required please create a new sheet.
