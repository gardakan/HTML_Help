Keyboard Navigation In A Single List Screen


 

A Single List Screen is one of the most common screens in Jazz. The screen is split into two main sections - a column-separated list occupies the upper part of the screen, and a bounce bar with various menu options occupies the bottom two rows of the screen. While the contents of each screen may vary, the keyboard controls are the same across most (if not all) of the List Screens in Jazz.
 
The Dual List Screen is a variant of the list view. More information can be found here.
