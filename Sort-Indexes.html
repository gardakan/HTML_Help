Using Sort Indexes On A List Screen


 

On some list view screens there will be the option to sort results using an index. In these cases there are options on the bounce bar whose names correspond to column headings. Selecting one of these options from the bounce bar will set it as the sort index, meaning the list view will sort the results first numerically lowest to highest, then alphabetically from a-z according to the selected index. The corresponding column will move to the left-most position, if it is not there already.



In the example below, you can see the cursor in the bounce bar highlighting the Company index menu option. (As this is the default sort order, the Company list column is already in the left-most position so selecting this option will not change the list display).

If CustNo sort menu option is selected, the sort index will change to order by customer number. The screenshot below illustrates that the Cust # column is now to the far left, and the results are sorted by lowest to highest value, numerically then alphabetically.
