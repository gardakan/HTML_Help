Using Filters


 

Filters are a way recall data based on a specific set of criteria. This guide describes the basic procedure for using filters the way they are implemented throughout Jazz.
NOTE: This guide assumes the user is familiar with basic keyboard navigation in a detail screen.

Filters are used to narrow the displayed data based on selected criteria. Filters are accessed via the bounce bar on a list screen.-

Highlighted filter option

Selecting the Filter option opens a grey pop-up window with various fields (typically drop-down menus and checkboxes) with criteria to use when searching for records. Select the fields you wish to search against and de-select the fields you wish to omit:

...and complete setting up the filter criteria by pressing enter when the cursor focus is on the Filter command at the bottom of the screen, after the last field on the filter form:

This will re-display the list view screen only with records matching the entered search criteria.

Selecting the Filter option again from the bounce bar will re-open the filter edit screen with the last active search criteria still selected (unless entering the screen after a reboot, in which case the default search criteria are selected). The filter criteria can be modified as often as is necessary, and will not impact the data it is searching.
