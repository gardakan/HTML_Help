Adding Items to a Price Sheet


 

From the price sheet detail view screen:

On the bounce bar, select the Items menu option

The Items pop-up screen will display any items already attached to the price sheet. A new Draft or Proposal will not have any. Move focus from the browse window to the bounce bar by pressing Tab.

Select the "Add" menu option from the bounce bar.

Select the item you wish to add from the drop-down menu.

Select the pricing units from the drop down menu.

If you require broker data to be entered before allowing orders for this item, check the Broker Data toggle with the spacebar.

Enter the price at levels 1-6 (if applicable).

The added item will be displayed on the item browse screen. To add more items, repeat the previous steps.
