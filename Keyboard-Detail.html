Keyboard Navigation In A Detail Screen


 

The Detail Screen is one of the most common screen types in Jazz. It represents a detailed, single-entry view of one record. The screen is divided into a top display section, where all the record information is shown. A bounce bar occupies the bottom two lines of the screen.

Screen behavior and modes

While different detail screens may have features that functions slightly differently from each other, for the most part they maintain very similar functionality. When navigating to a detail screen (usually from a List View screen), cursor focus starts on the bounce bar. Although there may be many menu options, almost every screen will have the options to Add a new record, skip back to the Previous record, advance to the Next record, and Change or Delete an existing record.

Detail screens have 3 different modes - Add mode, Edit mode and Display mode. Display mode is the default screen mode, in which the data fields show record information but cannot be edited. Add mode brings up an empty record with editable fields, and Edit mode opens the currently displayed record up for editing by a user.

There are six main types of data field - the Text Field, Drop-Down Menu, Checkbox, Radio Button, Date Field and Memo pop-up.
 Field Example  Description 

Text Field

   Text fields are general purpose data entry fields.  Most accept all alphanumeric characters as well as special characters, although there may be instances where the accepted data types are restricted. 

Drop-Down Menu

   Drop-Down Menus resemble text fields with a downward-pointing arrow at the right end.  These fields feature a navigable list of pre-selected options for the user to choose from.  While many drop-down menus only allow data selected from the drop-down list, there are some which will accept user-typed entries as well and in these cases, in addition to providing a list of values to choose from can function in the same way as a text field.

Checkbox

   Checkboxes provide 'yes' / 'no' data.  They have two states, 'off' and 'on'.

Radio Button

   A Rdio Button (or Radio Group) is a multiple-choice checkbox.  Usually, only one choice can be selected at a time.  Marking a second field as 'active' will automatically deactivate the first selected field.

Date Field

    A date field contains calendar data, usually in the standard US mm/dd/yy format.  Only numeric data following the accepted date format can be entered.  In certain cases there may be restrictions on entering dates outside of a range, or before the current date.  In these cases a pop-up message will alert the user what changes they need to make.

Memo pop-up

    The Memo Pop-Up is a way to access and write to database Memo Fields.  Entered data may use alphanumeric and special characters.

Unlike a text field, memo fields are suitable for longer entries and can be formatted across multiple lines into paragraphs.
