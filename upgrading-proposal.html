Upgrading a Proposal to a Price Sheet


 

Before a proposal can be upgraded to a price sheet it must have items attached to it.

Select the Approve bounce bar menu option.

Cursor focus is on the Allow Promo toggle. Press the spacebar to set it to "On".

Press enter. You should see a confirmation that the proposal has been approved.

On the bounce bar, select the Change menu option.

Move the cursor to the Price Sheet position and activate by pressing the spacebar.

You will see a pop-up window asking you if you want to continue. Select the "Yes" option.

The Price Sheet is now complete and can be attached to orders.
