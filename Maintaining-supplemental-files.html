Maintaining Supplemental Files


 

File maintenance refers to the direct process of adding, editing and deleting the data that is used in each module in Jazz. This page will describe the processes involved in maintaining files, and assumes that the user is familiar with keyboard navigation in a list view screen and a detail screen.

File maintenance in Jazz is performed in a similar manner across modules. Beginning at the main menu of each module, there is a menu option which leads to the maintenance section. From within the Sa and Po modules, generally the System Functions menu options:

S/A module main menu

P/O module main menu

...lead to the maintenance section:

S/A system functions menu

P/O system functions menu

...whereas in the Mp module, the Maintain Supplemental Files menu option

M/P Module Main Menu

...Will lead to the file maintenance menu:

M/P Supplemental Files Menu

Most menu options from these menus will lead to a list view screen displaying all of the records Jazz has for the selected category. Each selected category will also have a bounce bar with some or all of the options Add, Change, and Delete, as well as other options such as Print. The Add and Change options open a detail screen in an editable mode, whereas the Delete option has the same functionality as any other list screen delete option.

Sample file maintenance bounce bar.
NOTE: This information is general purpose - exceptions exist! If you need to enter data but are unable to follow steps outlined above, please check if there is specific documentation for the module you are working with.
